## Run on your machine  

### Prerequistes  

- Node.js v14^
- yarn

### Installation

1. Clone the project & Install packages from npm

```bash
$ git clone https://github.com/tmmoond8/noticon.git
$ cd noticon && yarn
```

2. Rename .env.sample to .env

3. Run

```bash
$ yarn dev
```
